####  坑爹虎  
##### 1#       楼主       发表于 2022-2-14 22:56

当代年轻人对中国传统武术的认知，大体上经历了这么一个跌宕起伏的过程。20世纪末，在李连杰、成龙等众多功夫明星的影响下，在徐克、袁和平等导演的电影作品里，传统武术披上了神秘的光环，成了驱除外辱、弘扬民族精神的图腾。仿佛只要学会了武术，就能成为无敌的高手，在擂台上战无不胜。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/9ee615f230f44d919d52147806ebae25?from=pc" height="453" id="aimg_BGV2r" onclick="zoom(this)" onmouseover="img_onmouseoverfunc(this)" style="cursor:pointer" width="800"/)电影《黄飞鸿》系列尽显传武之美
但到了千禧年之后，功夫片逐渐走向了没落。随着互联网的高速发展，过去只存在于江湖传说中的传武大师们，逐渐露出了真实的面目。尤其是在某打假狂人先后击败太极雷雷、田野等所谓的“传武大师”后，人们对传统武术实战能力的质疑不断增加。2020年，自诩传武大师的“闪电五连鞭”马保国，被不知名选手王庆民直接KO，从神坛跌落的传统武术，正在变成年轻人眼中的笑料。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/2edf74a644024ca4813dcda3a181e391?from=pc" height="311" id="aimg_N7pvo" onclick="zoom(this, this.src, 0, 0, 0)" width="554"/)

如今，这份笑料似乎正在迎来新的血液：自创无限制特种武道，却被网友调侃为“疯狗流”，但本人自称极具实战能力的传武高手陈鹤皋。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/3150dff3d3014dbdb55026691f0619ad?from=pc" height="1066" id="aimg_ruaMh" onclick="zoom(this)" onmouseover="img_onmouseoverfunc(this)" style="cursor:pointer" width="800"/)

这位姿势滑稽、手持剪刀的瘦削男子，就是陈鹤皋。因为他进行搏击的时候会浑身快速乱动，并发出如同狗叫的尖锐声音，就有了“疯狗”的绰号。陈鹤皋纵横江湖数十年，但却并未被某打假狂人上门挑战。某打假狂人曾说过这样一句话：“真的（传武高手），我打不过。我只能老老实实地打假。”那么，这位无限制特种武道开创者陈鹤皋，究竟是真是假呢？在此，我们不妨先来看看，无限制特种武道，到底是一种什么样的武术。和大众所认知的跆拳道、拳击等搏击技法不同，无限制特种武道完全是为街头巷斗而生的搏杀技法。1997年，陈鹤皋将自己的武术心得，编写到了《极厉害、实用的——无限制格斗术》一书中。这本书是无限制特种武道的内部资料，没有书号，也未曾正式出版发行，甚至还在封面上标注了“拒绝未成年者翻阅，禁止法制观念浅薄者学习”的警示语，市面上流通的并不多。但在上个世纪末，正值国内功夫热余温犹存的时期，在武术爱好者之间，这本书的名气不胫而走。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/b13013319e70498eac4a36de5cff16a1?from=pc" height="497" id="aimg_zHJ9s" onclick="zoom(this, this.src, 0, 0, 0)" width="353"/)

如果把《极厉害、实用的——无限制格斗术》放到武侠小说里，这本秘籍，会被自诩为名门正派的人士视为歪门邪道。引言里，陈鹤皋便强调，练习武术就是为了千方百计地保存自己，毫不留情地消灭敌人。为此，可以偷袭、暗算、群殴，总之就是要不择手段，甚至把敌人打死。和传统观念中“以德服人”“未习武，先习德”的那套说辞相比，陈鹤皋绝对是反面典型，堪比金庸笔下的魔教余孽。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/b3c4dab115114da683015fb25170c268?from=pc" height="652" id="aimg_EA293" onclick="zoom(this, this.src, 0, 0, 0)" width="430"/)

引言的最后，陈鹤皋强调，一切的前提都是合法，不能做违法斗殴行为。据传言，陈鹤皋传授弟子的第一门心法，就是《中华人民共和国刑法》（以下简称《刑法》），尤其是《刑法》中有关正当防卫的表述。而在无限制特种武道的门规里，陈鹤皋还特别强调了一点，门下弟子必须做到见义勇为，遇到歹徒勇敢上前制止。陈鹤皋说得神乎其神，但无限制特种武道真的能打死人吗？如果看了《极厉害、实用的——无限制格斗术》的话，那可能真的可以。一言以蔽之，这本书中所传授的技术，就是我们常说的“下三滥”的招数，插眼、踢裆、抓蛋，能够想到的阴招、狠招，这本书里应有尽有。为了进行招式说明，陈鹤皋亲笔绘画了演示图，惟妙惟肖，直观易懂。尤为有趣的是，陈鹤皋还给书中的小人配上了台词，四舍五入这本秘籍约等于一本漫画书。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/94045c5bbcde4a3eb8e9c54f49ea8f7c?from=pc" height="667" id="aimg_kpcCS" onclick="zoom(this, this.src, 0, 0, 0)" width="518"/)

静态图片的表现力终究有限，动态的演示，更能展现出无限制特种武道的精髓。后撩裆踢击，就是最能展现这门武术的招式之一。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/e52ef971f103432086100d10a86799d3?from=pc" height="270" id="aimg_Z1sS6" onclick="zoom(this, this.src, 0, 0, 0)" width="480"/)

无限制特种武道里，可以有一百种踢裆的方法。除了从后面踢，还可以从正面侧踢裆部。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/d393dc955f6a44bc9d22b83475cddb8d?from=pc" height="270" id="aimg_tmQR5" onclick="zoom(this, this.src, 0, 0, 0)" width="480"/)

在将敌人击倒在地后，更是可以从上向下踩踏裆部，看着都疼。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/3d37a6f732c9454a913406d6558a49a9?from=pc" height="270" id="aimg_T399F" onclick="zoom(this, this.src, 0, 0, 0)" width="480"/)

而在组合攻击中，攻击裆部与插眼可以连环发动，阴损至极。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/8638bdac54d540da8b2319c970898f2a?from=pc" height="270" id="aimg_uJ0gj" onclick="zoom(this, this.src, 0, 0, 0)" width="480"/)

以上，只是无限制特种武道的冰山一角。但有一点可以确认，这个门派的招式，招招都是奔着敌人的要害去的，被打到的敌人非死即伤。这也就不难理解，为什么陈鹤皋要求门人先研究《刑法》。而且他特意强调，绝不能拿这些招式进行街头斗殴，必须为正义而战，尤其不能用来攻击普通群众。只有在面对生命受到威胁的时候，才能下杀手。对此，可能有人会产生怀疑，传武都是花拳绣腿，那些成名已久的大师都接连翻车，陈鹤皋这一套功夫说得惊悚吓人，但在实战里，真能杀人，而不是被杀吗？遗憾的是，并没有陈鹤皋本人的实战视频公之于众，我们并不能知道这位大师的实战能力究竟如何。但幸运的是，陈鹤皋的弟子，却留下了众多极具代表性的实战记录，可以佐证无限制特种武道的实战能力。早年间，陈鹤皋的大弟子张泽中，意外与11名暴徒当街冲突，在1对11的悬殊人数比下，张泽中当场打死1名暴徒、重伤4名暴徒。其余暴徒见状转身就跑，张泽中继续追击，又打伤了几名暴徒。事后，公审人员在对相关责任进行判罚时，判定张泽中杀人属于正当防卫。但是，他在暴徒逃跑后继续追击的行为，被判定为防卫过当。但即便如此，张泽中也无需对此事承担刑事责任，只是赔偿了一笔医疗费。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/5e53957238cd46bb91730073c6da2ff3?from=pc" height="629" id="aimg_O8GO8" onclick="zoom(this, this.src, 0, 0, 0)" width="678"/)

或许正是因为这次事件，陈鹤皋加强了对《刑法》中正当防卫的研究，目的可以说相当明显——如何做到打人甚至打死人，而自己无需承担刑事责任。在陈鹤皋的众多弟子中，张泽中并非唯一一个杀过人的。2001年，家住广西玉林的陈鹤皋弟子冯健汉，被十多个恶霸上门滋事，双方言语冲突中，几位恶霸对冯健汉的家人大打出手。冯健汉奋起反击，徒手打死一人，打伤二人，吓退其余歹徒。玉林市公安局对此事作出判罚，冯健汉属于正当防卫，无需为杀人承担刑事责任。同时，上门滋事的恶霸，需赔偿冯健汉一家医药费等损失。2002年，陈鹤皋弟子刘艳，在东莞火车站与三名流氓产生口角。随后，流氓又叫来四名同伙，手持西瓜刀与短棍围攻刘艳。冲突中，刘艳仅凭一把剪刀，当场打死一名流氓，重伤三人，成功逃跑。后续判决中，刘艳被判正当防卫，无需承担刑事责任。2005年，陈鹤皋弟子罗神贵，与一名女性友人搭乘公交车。途中，三名小偷对这位女性友人的手机进行盗窃，罗神贵发现后大声喝止。小偷怀恨在心，在罗神贵两人下车后追上去堵截，其中一名小偷手持匕首，一名手持长棍。冲突中，罗神贵同样拿出了师门经典兵器剪刀，当场击杀一名小偷，并将另外两名小偷打成重伤。事后，深圳检察院判定罗神贵属于正当防卫，无需承担刑事责任。而这起事件较为典型，因此作为正当防卫的案例，登上了央视的《今日说法》与深圳卫视的《城市记录》等节目。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/951ede419b094464ab866a2ed5c0b22f?from=pc" height="363" id="aimg_FzCCR" onclick="zoom(this, this.src, 0, 0, 0)" width="629"/)

作为被罗神贵反杀的人，受伤的小偷在接受采访时痛哭流涕，十分委屈地表示，罗神贵突然在他们面前浑身乱抖，疯狂乱叫，他们还以为这个人狂犬病发作了。2005年，成了一个分水岭。在此之前，堪称行为艺术大师的陈鹤皋，早已被当时的广大武术爱好者口诛笔伐。他们认为不讲武德的无限制特种武道，是在玷污、抹黑中国武术。他们骂陈鹤皋是疯狗，嘲笑他是武林败类。

而在罗神贵事件公开后，诋毁陈鹤皋的人慢慢消失了。与之相对的，每当有人质疑无限制特种武道的实战能力，网友都会拿一件件真实案例进行反驳。尤为重要的是，陈鹤皋的这些弟子，杀的都是歹徒、恶霸、小偷，简直是当代社会的替天行道。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/b4e7ccf11b3c4e03893ff983195546d2?from=pc" height="397" id="aimg_IW4BZ" onclick="zoom(this, this.src, 0, 0, 0)" width="544"/)

在此稍作科普，世纪之交的那段时期，中国正处在经济高速发展、快速摆脱贫困的社会激荡中。在此期间，社会上的偷窃、抢劫等违法案件不断上演，尤其是在偏远地区，治安状况较差，各种社会闲杂人等聚集在车站、广场等场合，屡屡寻衅滋事。正是在这样的大背景下，武术这门古老的技术，受到了当时年轻人的追捧。武术热的背后，有着人民增强自卫能力、抗击歹徒的现实需求。和传统的套路武术相比，陈鹤皋的无限制特种武道，展现出了强大的实战能力。随着国家社会治安状况不断改善，扫黑除恶工作的有力推进，无限制特种武道也在逐渐失去实战的舞台。对于大众而言，这当然是一件好事。与此同时，无限制特种武道也在其他的领域，继续发光发热。2009年，陈鹤皋担任武警浙江省总队反恐特战格斗指导教官和某监狱防暴大队警务技能教官。2016年，陈鹤皋被武警浙江省总队聘请为反恐特战格斗技能总指导。无限制特种武道已经进入到了国家反恐体系之中。这一履历，足以让陈鹤皋与那些招摇撞骗的假传武大师，划清界限。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/471796d2f37f41ea87eb334e3cbc308c?from=pc" height="377" id="aimg_X1Fz4" onclick="zoom(this, this.src, 0, 0, 0)" width="552"/)

近年来，在互联网的快速传播下，陈鹤皋及其门下弟子的事迹，被更多的网友所熟知。如今，已经年过60岁的陈鹤皋，仍然在网络上分享着简单实用的自卫技巧。时至今日，网友依然会用“疯狗拳”来调侃这位老人家，但在今天的互联网语境下，这种调侃早已不是辱骂。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/5736969b7f2e4fe9b726716cb47390c7?from=pc" height="417" id="aimg_YfhvI" onclick="zoom(this, this.src, 0, 0, 0)" width="284"/)

同时，随着各类搏击技法的普及，人们对陈鹤皋如同疯狗一样的啸叫，也有了不一样的理解。不难发现，陈鹤皋的狗叫，和知名武术家李小龙在电影中标志性的啸叫极为相似。对战中，啸叫不仅可以起到扰乱对手心神、缓解自身紧张的作用，还能刺激肾上腺素，爆发出更为强大的力量。这种技法，在跆拳道等技击术中同样存在。在嘲笑陈鹤皋是疯狗的时候，又有多少人知道，啸叫背后的科学道理呢？只是必须承认，陈鹤皋的武术，是极不适合观赏的——除非你是抱着找乐子的心态。<img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/9092f469dddf499fa949d7d792a06b2d?from=pc" height="267" id="aimg_ne9eA" onclick="zoom(this, this.src, 0, 0, 0)" width="480"/)

但是，陈鹤皋的这份滑稽，才是传统武术的真实形态。比起那些坐而论道、空有理论的“武术大师”，谁才是真正的滑稽，已经不言自明。以及，永远不要忘记，陈鹤皋在传授弟子的时候，第一门课程是《刑法》<img src="https://static.saraba1st.com/image/smiley/face2017/067.png" referrerpolicy="no-referrer">。
